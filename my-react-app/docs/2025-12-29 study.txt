useDispatch와 useAppDispatch의 차이 공부
useDispatch -> 원본 Hook (타입 정보 없음 / 약함)
useAppDispatch -> 우리가 만든 타입이 박힌 Hook

dispatch는 단순히 reducer를 직접 호출하는것이 아님
reducer는 store 안에 등록된 함수고 dispatch는 액션을 처리해 달라는 요청을 한다 ( dispatch 안에 Slice에서 생성된 action creator를 호출해 (Slice name)/(reducer 이름) 형태의 action 객체 생성 )
dispatch는 action을 store로 전달하고, store에 등록된 reducer들이 action.type을 기준으로 "자기 액션"인지 판단한다
store.ts는 App 최상단에 Provider로 등록되어 있어 React 트리에 주입되고 그 트리의 store.ts를 dispatch와 selector가 사용함
즉, 논리적으로 반드시 dispatch와 selector는 반드시 store.ts를 경유한다.
